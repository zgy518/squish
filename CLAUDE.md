## 全局约束

1. **零预算**：所有服务必须用免费层
2. **纯浏览器端优先**：核心计算在客户端完成，不依赖付费 API
3. **英文界面**：目标用户是海外英语用户
4. **渐进开发**：每次只做一步，验证后再继续
5. **用户操作归用户**：注册账号、配置服务等由用户自己完成，AI 负责代码

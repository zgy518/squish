## 变现模型

```
免费层：单文件≤10MB，每日15次压缩，标准质量压缩
付费层：$9.99 终身，无限压缩，单文件≤50MB，目标大小压缩，格式转换，批量处理
```

收款链路：
```
用户 → LemonSqueezy 支付 → License Key → 网站验证 → 解锁
```

---

## 非功能性需求

- **性能**：单张压缩 2 秒内完成，首屏加载 < 2s
- **隐私**：所有处理在浏览器端完成，图片不上传服务器
- **兼容性**：支持 Chrome/Firefox/Safari/Edge 近 2 年版本
- **SEO**：Semantic HTML、结构化数据、sitemap、meta tags
